MaterialAnimationBuilder

Avatar内に含まれる複数のMaterialのProperty値をまとめて変更するAnimationClipの作成を補助するツールです。

使い方
1.Scene上に「MaterialAnimationBuilder」Componentを設定したGameObjectを設置してください。
2.AnimationのRootとなるObjectを「RootObject」に設定してください。
3.「RootObjectからMaterialを取得」ボタンを押下し、Material情報を取得します。
4.「設定先のAnimationClip」に書き込み対象のAnimationClipを設定してください。
5.「Animation開始時間」「Animation終了時間」「Property」に関する項目を設定してください。
6.「AnimationClipへの書き込み」
